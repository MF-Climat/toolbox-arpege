# Toolbox-ARPEGE


## Getting started

This is a Toolbox for ARPEGE-CLIMAT. The aim is to provide tools usefull to prepare and debug a configuration of ARPEGE-CLIMAT
The repository is not intended to host your work. Clone it, copy the script you want on your workspace and modify it as much as you want.


## Current state

Geometry TL127 has been tested on belenos.
Geometry TL49 is coming soon.


## Toy

It provides very simple script to run the model. Data necessary to run the model should be available in a rundir from ECLIS.
Data are also available on DAAC-DEV directory on belenos + repository on Lustre.


## PGD-PREP

It provide a script to generate a PGD and a PREP. 


## Namelists

Namelists are prepared to be used in toys scripts i.e. : some keys are set to AUTO or with variables modified by the scripts in toy directory.
